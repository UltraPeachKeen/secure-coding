# 기능 테스트 체크리스트

수동 테스트 시 날짜, 실행 환경, 테스터와 실제 결과를 기록하고 중요한 화면을 캡처한다.

| ID | 테스트 | 기대 결과 | 자동 테스트 |
|---|---|---|---|
| F-01 | 신규 아이디 가입 | 가입 성공, 초기 잔액 생성 | O |
| F-02 | 같은 아이디 재가입 | 중복 오류 | O(폼 검증 포함) |
| F-03 | 정상/오류/휴면 로그인 | 정상만 로그인 | O |
| F-04 | 프로필·소개 수정 | 저장 후 공개 프로필 반영 | O |
| F-05 | 현재 비밀번호 확인 후 변경 | 변경 후 재로그인 요구 | O |
| F-06 | 상품 등록/상세/검색 | 목록·검색·상세에 표시 | O |
| F-07 | 본인 상품 수정/삭제 | 수정 반영, 삭제 후 공개 숨김 | O |
| F-08 | 전체 채팅 전송/polling | 전체 방에 새 메시지 표시 | O(서버) / 수동(UI) |
| F-09 | 1:1 채팅 | 두 사용자에게만 표시 | O |
| F-10 | 상품/사용자 신고 | 사유와 신고자 저장 | O |
| F-11 | 서로 다른 3명 신고 | 상품 차단/사용자 휴면 | O |
| F-12 | 정상 송금 | 두 잔액과 내역 동시 변경 | O |
| F-13 | 관리자 사용자/상품 상태 변경 | 상태와 감사 로그 반영 | O |
| F-14 | 관리자 신고·거래 조회 | 전체 내역 표시 | 수동 |
| F-15 | `seed-demo` | 관리자·일반 계정·상품 생성, 재실행 안전 | CLI 검증 |
